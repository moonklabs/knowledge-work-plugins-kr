# Roadmap Update

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

로드맵 변경 요청을 반영해 우선순위와 일정안을 업데이트합니다.

## Workflow

1. 변경 입력(목표, 제약, 리소스) 확인
2. 항목 점수화(RICE 등) 및 의존성 점검
3. Now/Next/Later 또는 분기 로드맵 재배치
4. 영향도(지표, 리스크, 커뮤니케이션) 정리

## Output

- Updated roadmap view
- Priority rationale
- Trade-off notes
- Stakeholder message draft
