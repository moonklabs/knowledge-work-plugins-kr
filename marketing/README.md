# Marketing 플러그인

[Cowork](https://claude.com/product/cowork)를 위한 마케팅 플러그인입니다. Anthropic의 에이전트형 데스크톱 앱인 Cowork에 맞춰 설계했지만 Claude Code에서도 동작합니다. 콘텐츠 제작, 캠페인 기획, 브랜드 보이스 관리, 경쟁 분석, 성과 리포팅을 지원합니다.

## 설치

```bash
claude plugins add knowledge-work-plugins/marketing
```

## 커맨드

| Command | 설명 |
|---|---|
| `/draft-content` | 블로그, 소셜 미디어, 이메일 뉴스레터, 랜딩 페이지, 보도자료, 케이스 스터디 초안 작성 |
| `/campaign-plan` | 목표, 채널, 콘텐츠 캘린더, 성공 지표가 포함된 캠페인 브리프 생성 |
| `/brand-review` | 브랜드 보이스, 스타일 가이드, 메시지 기둥에 따라 콘텐츠 리뷰 |
| `/competitive-brief` | 경쟁사 조사 및 포지셔닝/메시지 비교 생성 |
| `/performance-report` | 핵심 지표, 트렌드, 최적화 제안을 포함한 마케팅 성과 리포트 생성 |
| `/seo-audit` | 키워드 리서치, 온페이지 분석, 콘텐츠 갭, 기술 체크, 경쟁사 비교를 포함한 SEO 감사 |
| `/email-sequence` | 너처 플로우, 온보딩, 드립 캠페인 등 멀티 이메일 시퀀스 설계 및 작성 |

## 스킬

| Skill | 설명 |
|---|---|
| `content-creation` | 콘텐츠 유형별 템플릿, 채널별 작성 모범 사례, SEO 기본, 헤드라인 공식, CTA 가이드 |
| `campaign-planning` | 캠페인 프레임워크, 채널 선택, 콘텐츠 캘린더 생성, 예산 배분, 성공 지표 |
| `brand-voice` | 브랜드 보이스 문서화, 보이스 특성, 톤 조정, 스타일 가이드 적용, 용어 관리 |
| `competitive-analysis` | 경쟁 조사 방법론, 메시징 비교, 콘텐츠 갭 분석, 포지셔닝, 배틀카드 생성 |
| `performance-analytics` | 채널별 핵심 지표, 리포팅 템플릿, 트렌드 분석, 어트리뷰션 모델링, 최적화 프레임워크 |

## 예시 워크플로

### 블로그 글 초안 작성

```
> /draft-content
Type: blog post
Topic: AI가 B2B 마케팅을 어떻게 변화시키는가
Audience: 미드마켓 SaaS 기업의 마케팅 디렉터
Key messages: AI는 반복 업무 시간을 줄이고, 개인화를 높이며, 인간의 검토가 필요함
Tone: 권위 있지만 접근하기 쉬운 톤
Length: 1200 words
```

Claude는 매력적인 헤드라인, 훅이 있는 인트로, 구조화된 섹션, SEO 최적화된 소제목, 명확한 CTA가 포함된 블로그 글 초안을 생성합니다.

### 캠페인 계획

```
> /campaign-plan
Goal: 신제품 런칭에서 500명 가입 달성
Audience: 엔터프라이즈 기업의 기술 의사결정자
Timeline: 6주
Budget range: $20,000-$30,000
```

Claude는 목표, 대상 세그먼트, 핵심 메시지, 채널 전략, 주차별 콘텐츠 캘린더, 추적할 KPI를 포함한 캠페인 브리프를 제공합니다.

### 브랜드 가이드라인 기반 리뷰

```
> /brand-review
[작성한 콘텐츠 붙여넣기]
```

로컬 설정에 브랜드 스타일 가이드가 구성되어 있으면 Claude가 보이스, 톤, 용어, 메시지 기둥 기준으로 콘텐츠를 점검합니다. 구성되어 있지 않으면 가이드라인을 물어보거나 명확성, 일관성, 전문성을 기준으로 일반 리뷰를 제공합니다.

## 설정

로컬 설정 파일에 브랜드 보이스, 스타일 가이드, 타겟 페르소나를 구성하면 개인화된 결과를 받을 수 있습니다. 이렇게 하면 `/draft-content`, `/brand-review` 같은 커맨드가 매번 묻지 않고 브랜드 기준을 자동 적용합니다.

## MCP 통합

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](CONNECTORS.md)를 보세요.

이 플러그인은 다음 MCP 서버와 연동됩니다.

- **Slack** — 팀과 드래프트, 리포트, 브리프 공유
- **Canva** — 디자인 에셋 생성 및 편집
- **Figma** — 디자인 파일과 브랜드 에셋 접근
- **HubSpot** — 캠페인 데이터 가져오기, 연락처 관리, 마케팅 자동화 추적
- **Amplitude** — 성과 리포팅을 위한 제품 분석/행동 데이터
- **Notion** — 브리프, 스타일 가이드, 캠페인 문서 접근
- **Ahrefs** — SEO 키워드 리서치, 백링크 분석, 사이트 감사
- **Similarweb** — 경쟁 트래픽 분석 및 시장 벤치마킹
- **Klaviyo** — 이메일 마케팅 시퀀스/캠페인 작성 및 리뷰
