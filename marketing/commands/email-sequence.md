# Email Sequence

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

목적별 이메일 시퀀스를 설계하고 각 메일 카피를 생성합니다.

## Trigger

```bash
/email-sequence [type]
```

## Workflow

1. 시퀀스 목적 정의: 전환, 활성화, 리텐션, 재참여
2. 분기 설계: 트리거, 대기일, 분기 조건, 종료 조건
3. 메시지 작성: 메일별 목표/제목/본문/CTA
4. 성과 설계: 오픈/클릭/전환 KPI와 개선 가설

## Output

- Sequence map
- Email-by-email copy
- Timing plan
- KPI checklist
