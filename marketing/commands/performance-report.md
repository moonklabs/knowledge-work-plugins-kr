# Performance Report

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

기간/캠페인 성과를 요약하고 실행 가능한 최적화 제안을 제공합니다.

## Trigger

```bash
/performance-report <period>
```

## Workflow

1. 데이터 수집: 채널별 트래픽, 전환, 비용, 매출/기여도
2. 지표 정리: CAC, ROAS, CVR, CTR, 파이프라인 기여
3. 추세 분석: 전기 대비 증감 및 이상치 확인
4. 인사이트 도출: 성과 드라이버/비효율 원인 추정
5. 액션 제안: 즉시 실행/실험 항목 구분

## Output

- Exec summary
- KPI table
- Channel insights
- Optimization plan
