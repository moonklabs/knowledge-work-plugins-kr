# Draft Content

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

콘텐츠 유형과 대상에 맞는 마케팅 초안을 작성합니다.

## Trigger

```bash
/draft-content <type> <topic>
```

## Workflow

1. 브리프 확인: 대상, 목표, 톤, 길이, CTA, 금지 표현 확인
2. 아웃라인 작성: 핵심 주장, 근거, 섹션 구조
3. 초안 생성: 채널 맞춤 톤/형식 반영
4. 품질 점검: 명확성, 반복, 브랜드 일관성, 사실성 점검
5. 최종 전달: 본문 + 대체 제목/CTA + 변형안 제공

## Output

- Draft v1
- Title options
- CTA options
- Quick edit notes
