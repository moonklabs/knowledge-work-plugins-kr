# /seo-audit

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

사이트 SEO 상태를 점검하고 우선순위 실행 계획을 제공합니다.

## Trigger

```bash
/seo-audit <url or topic>
```

## Workflow

1. 범위 정의: 도메인/섹션/키워드 묶음
2. 기술 점검: 인덱싱, 속도, 메타, 구조화 데이터, 내부링크
3. 온페이지 점검: 제목/헤딩/콘텐츠 품질/검색의도 적합성
4. 키워드/경쟁 분석: 격차 키워드와 난이도/기회 평가
5. 우선순위화: Impact x Effort 기준으로 정렬

## Output

- Audit scorecard
- Critical issues
- Quick wins
- 30/60/90 day plan
