# Competitive Brief

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

경쟁사 포지셔닝, 메시지, 콘텐츠 전략을 비교 분석합니다.

## Trigger

```bash
/competitive-brief <competitor>
```

## Workflow

1. 비교 범위 정의: 경쟁사, 제품군, 비교 축을 확정합니다.
2. 증거 수집: 사이트, 랜딩, 가격, 콘텐츠, 캠페인 신호를 수집합니다.
3. 비교 매트릭스: 가치제안, 톤, CTA, 채널, 차별점/약점을 표로 정리합니다.
4. 시사점 도출: 우리가 취할 메시지/채널 우선순위를 제안합니다.

## Output

- Executive summary
- Comparison matrix
- Threats & opportunities
- Recommended actions
