# 어카운트 리서치

아웃리치 전에 모든 기업 또는 인물에 대한 완전한 그림을 제공합니다. 이 스킬은 웹 검색만으로도 항상 작동하며, 데이터 보강 및 CRM 데이터가 추가되면 크게 향상됩니다.

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                     ACCOUNT RESEARCH                             │
├─────────────────────────────────────────────────────────────────┤
│  기본 기능 (웹 검색으로 단독 작동)                                  │
│  ✓ 기업 개요: 사업 내용, 규모, 산업                                │
│  ✓ 최신 뉴스: 투자, 리더십 변경, 발표                              │
│  ✓ 채용 신호: 채용 공고, 성장 지표                                 │
│  ✓ 주요 인물: LinkedIn 기반 리더십 팀                              │
│  ✓ 제품/서비스: 판매 제품, 대상 고객                               │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + 데이터 보강: 검증된 이메일, 전화번호, 기술 스택, 조직도           │
│  + CRM: 기존 관계, 과거 영업 기회, 연락처                          │
└─────────────────────────────────────────────────────────────────┘
```

---

## 시작하기

조사 대상을 알려주세요:

- "Research Stripe"
- "Look up the CTO at Notion"
- "Intel on acme.com"
- "Who is Sarah Chen at TechCorp?"
- "통화 전에 [회사]에 대해 알려줘"

즉시 웹 검색을 실행합니다. 데이터 보강이나 CRM이 연결되어 있으면 해당 데이터도 가져옵니다.

---

## 커넥터 (선택사항)

도구를 연결하여 이 스킬을 강화하세요:

| 커넥터 | 추가 기능 |
|--------|----------|
| **데이터 보강** | 검증된 이메일, 전화번호, 기술 스택, 조직도, 투자 상세 정보 |
| **CRM** | 기존 관계 이력, 과거 영업 기회, 기존 연락처, 메모 |

> **커넥터가 없나요?** 문제없습니다. 웹 검색만으로도 모든 기업 또는 인물에 대한 충실한 리서치를 제공합니다.

## 실행 흐름

### 1단계: 요청 분석

```
조사 대상 파악:
- "Research Stripe" → 기업 리서치
- "Look up John Smith at Acme" → 인물 + 기업
- "Who is the CTO at Notion" → 직무 기반 검색
- "Intel on acme.com" → 도메인 기반 조회
```

### 2단계: 웹 검색 (항상 실행)

```
다음 검색 실행:
1. "[기업명]" → 홈페이지, 소개 페이지
2. "[기업명] news" → 최신 발표
3. "[기업명] funding" → 투자 이력
4. "[기업명] careers" → 채용 신호
5. "[인물명] [기업] LinkedIn" → 프로필 정보
6. "[기업명] product" → 판매 제품
7. "[기업명] customers" → 고객층
```

**추출 정보:**
- 기업 설명 및 포지셔닝
- 최신 뉴스 (최근 90일)
- 리더십 팀
- 채용 공고
- 기술 관련 언급
- 고객층

### 3단계: 데이터 보강 (연결 시)

```
데이터 보강 도구 사용 가능 시:
1. 기업 보강 → 기업 정보, 투자, 기술 스택
2. 인물 검색 → 조직도, 연락처 목록
3. 인물 보강 → 이메일, 전화번호, 경력
4. 신호 수집 → 구매 의향 데이터, 채용 속도
```

**데이터 보강 추가 정보:**
- 검증된 연락처 정보
- 완전한 조직도
- 정확한 직원 수
- 상세 기술 스택
- 투자자 포함 투자 이력

### 4단계: CRM 확인 (연결 시)

```
CRM 사용 가능 시:
1. 도메인으로 어카운트 검색
2. 관련 연락처 조회
3. 영업 기회 이력 조회
4. 활동 타임라인 조회
```

**CRM 추가 정보:**
- 기존 관계 맥락
- 과거 사례 (성공/실패 거래)
- 기존 대화 상대
- 메모 및 이력

### 5단계: 종합

```
1. 모든 소스 결합
2. 데이터 보강 정보를 웹 정보보다 우선 (더 정확)
3. CRM 맥락 추가 (존재 시)
4. 자격 심사 신호 식별
5. 대화 포인트 생성
6. 접근 방식 권고
```
