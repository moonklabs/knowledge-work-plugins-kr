# customer-research 스킬

이 스킬은 'customer-support/skills/customer-research' 범위의 작업을 일관된 절차로 수행하기 위한 지침입니다.

## 목적

- 요청 의도를 빠르게 명확화
- 필요한 입력과 제약을 먼저 확인
- 재현 가능한 산출물 생성

## 입력

- 사용자 목표/질문
- 관련 데이터 또는 문서
- 기간, 범위, 우선순위

## 절차

1. 요구사항과 성공 기준을 한 줄로 정리합니다.
2. 사용 가능한 연결 소스와 로컬 파일을 확인합니다.
3. 핵심 작업을 단계별로 수행합니다.
4. 불확실성, 가정, 리스크를 명시합니다.
5. 결과물을 사용 가능한 형식으로 제공합니다.

## 출력

- 핵심 결과 요약
- 근거(데이터/출처/가정)
- 후속 액션 제안

## 품질 기준

- 모호한 표현 없이 구체적으로 작성
- 불필요한 장문 설명 지양
- 사용자 의사결정에 필요한 정보 우선 제시
