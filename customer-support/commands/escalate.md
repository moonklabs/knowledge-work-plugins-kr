# Escalate

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

지원 이슈를 내부 대응팀이 바로 처리할 수 있는 에스컬레이션 브리프로 정리합니다.

## Usage

```bash
/escalate <issue>
```

## Workflow

1. 이슈/재현 단계/영향 고객 수집
2. 비즈니스 영향(ARR, SLA, 확산 가능성) 평가
3. 원인 가설 및 필요한 팀 정의
4. 요청사항(조치/ETA/오너) 명시

## Output

- Escalation brief
- Severity and impact
- Reproduction checklist
- Requested actions and owners
