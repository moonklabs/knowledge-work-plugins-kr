# Draft Response

> 낯선 플레이스홀더가 보이거나 어떤 도구가 연결되어 있는지 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 보세요.

상황/긴급도/고객 상태에 맞춰 고객 응답 초안을 작성합니다.

## Usage

```bash
/draft-response <situation>
```

## Workflow

1. 맥락 수집(이슈, 감정 상태, 기대치, SLA)
2. 톤 선택(사과, 안내, 상태 공유, 해결 제안)
3. 본문 작성(인정, 원인/현황, 다음 단계, 일정)
4. 법무/보안 민감 표현 점검

## Output

- Customer-facing response
- Short variant
- Internal notes(optional)
